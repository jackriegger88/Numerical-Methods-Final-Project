# Numerical-Methods-Final-Project
Project Title
Numerical Analysis of a Three-Species Food Chain using the Hastings-Powell Model

Description
This project simulates the population dynamics of a three-level food chain (Resource, Consumer, and Top Predator). Unlike simpler two-species models, a three-species system can exhibit chaotic behavior and complex oscillations. I will use the 4th-order Runge-Kutta (RK4) method to numerically integrate the coupled differential equations governing the system. The goal is to determine the "stability zones" where all three species coexist without any reaching extinction.
